# Collaborate.io

A real-time collaborative text writing platform built with the MERN stack (Express, React, Node, and Socket.io).

## Features
- **Real-time Synchronization**: Multiple users can write and see changes instantly.
- **Premium Design**: Sleek dark mode with glassmorphism and smooth animations.
- **Connection Status**: Live indicator showing server connection state.
- **No Database Needed**: Uses server-side memory for simplicity (beginner-friendly).

## How to Run

### 1. Start the Server
```bash
cd server
npm install
npm run start (or nodemon server.js)
```

### 2. Start the Client
```bash
cd client
npm install
npm run dev
```

### 3. Open in Browser
Visit `http://localhost:5173` in multiple browser tabs to see the real-time collaboration in action!
